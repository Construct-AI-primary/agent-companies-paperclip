## Memory routing
- Use retrieval, not preload.
- Durable facts -> PARA (`~/life/`).
- Daily residue -> `memory/YYYY-MM-DD.md`.
- Session recovery -> LCM / lossless-claw.
- Cross-session graph recall -> OpenStinger (optional).
- Auto-recall + capture -> Gigabrain.
