It details company profiles, team structures, agent rosters, interdependencies, collaboration patterns, and integration points to enable effective task assignment and team coordination.

## Runtime: Hermes Agent

All companies use the **Hermes agent** runtime via the `hermes_local` adapter in Paperclip. Key capabilities:

| Capability | Details |
|------------|---------|
| **Adapter** | `hermes_local` (runs Hermes CLI locally) |
| **Model** | OpenRouter (e.g. `openrouter/anthropic/claude-sonnet-4.5`) |
| **Session** | Persistent (`persistSession: true`) |
| **Toolsets** | `file_tools`, `terminal_tools`, `code_execution` (varies by agent) |
| **Memory** | SQLite with FTS5 search, context compression |
| **Messaging Gateway** | Telegram, Discord, Slack, WhatsApp, Signal |
| **Profiles** | Multiple isolated Hermes instances per company |
| **Skills** | Per-company skill packages (one per agent) |
