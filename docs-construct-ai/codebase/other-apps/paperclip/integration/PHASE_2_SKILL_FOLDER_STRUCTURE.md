# Phase 2: Agent Skill Folder Structure

## Overview

This document defines the complete folder structure for all 170+ agent skills across 6 companies. Each folder follows the naming convention:
```
agentname-company-role-description
```

This ensures:
- No naming conflicts between companies
- Clear identification of agent origin
- Easy discovery and routing
- Enterprise-scale organization
