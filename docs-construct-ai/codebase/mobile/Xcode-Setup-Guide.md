## 📱 TROUBLESHOOTING & QUICK FIXES

### Issue: "Simulator not launching"
```bash
# Kill all processes
killall Simulator
killall "iOS Simulator"

# Reset simulator
xcrun simctl shutdown all
xcrun simctl erase all

# Restart Xcode and try again
npm run ios
```

### Issue: "Camera permissions error"
```bash
# In Simulator
Hardware → Reboot Simulator
# OR: Reset settings
Hardware → Erase All Content and Settings
```

### Issue: "Metro bundler not connecting"
```bash
# Clear cache and restart
npm start -- --clear

# Hard reset
rm -rf node_modules
npm install
npm run ios
```
