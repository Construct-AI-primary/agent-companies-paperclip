Project Starter

Click a button to copy the exact command. Paste it in your VSCode terminal (PowerShell 7). No special tooling required.

1) Initiate (install dependencies)

Installs dependencies at root, client, and server.

npm install && cd client && npm install && cd .. && cd server && npm install && cd ..
Run in project root: C:\Users\Admin\Documents\GitHub\construct_ai\construct_ai

2) Build & Serve – Client

Starts the React dev server for the client.

cd client && npm run dev
Default dev server: http://localhost:3001/

3) Build & Serve – Server

Starts the API/server dev process.

cd server && npm run dev
Default API proxied by client at /api → http://localhost:3003/

4) Build & Serve – Both

Opens two terminals: one for server and one for client (PowerShell 7).

Start-Process pwsh -ArgumentList '-NoExit','-Command','cd server; npm run dev' -WindowStyle Normal; Start-Process pwsh -ArgumentList '-NoExit','-Command','cd client; npm run dev' -WindowStyle Normal
Run in project root. This launches two new terminal windows.

5) Modal Test Route

Select only the page/discipline here. The App will list all available modals for that page. 

http://localhost:3001/#/modal-test
This opens: http://localhost:3001/#/modal-test?discipline=.... The in‑app selector will list the modals.
