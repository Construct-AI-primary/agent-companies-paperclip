# 02025 Cross-Discipline Measurement Platform AI-Native Operations Prompt Template

## Overview

This AI-Native development prompt guides Paperclip AI agents (KnowledgeForge AI, LearningForge AI, DomainForge AI, IntegrateForge AI) to implement cross-discipline measurement platform features including measurement, valuations, cost control, and final accounts. **This prompt is for Paperclip AI agents operating in the cross-discipline measurement ecosystem.**

## Implementation Phases

### Phase 1: Foundation — Data Architecture
- [ ] Classify data: text-native (contracts, correspondence) or structured (BOQ, valuations, payment registers)
- [ ] Implement BOQ extraction and measurement calculation pipeline

### Phase 2: Document Generation
- [ ] Template library — Interim valuations, final accounts, variation orders, measurement sheets

### Phase 3: Multi-Agent QS
- [ ] Agents: QS Manager, Quantity Surveyor, Estimator, Final Accounts Specialist

### Phase 4: Cost Intelligence
- [ ] Cost forecasting, earned value analysis, variation trend analysis

### Phase 5: AI Safety Boundaries
- [ ] Non-delegable: Final account certification requires human sign-off
- [ ] AI must disclose measurement assumptions

## Automation Spectrum
| Level | Examples |
|-------|----------|
| Full Automation | BOQ formatting, measurement sheet generation, cost register updates |
| Augment | Interim valuation drafting, variation cost assessment |
| Human-Led | Final account certification, dispute settlement, cost advice |
