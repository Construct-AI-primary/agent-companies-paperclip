# Energy & Power Project Procurement Typology

## Sector Overview
Energy & power projects include utility-scale solar PV and CSP, onshore and offshore wind, thermal power stations, hydroelectric dams, transmission lines, and substations. Procurement involves specialized equipment with strict technical specifications, grid integration requirements, and high-value service contracts.

## Common PO Types (Purchase Orders for Goods)

| PO Category | Typical Goods | Key Specifications | Lead Times |
|-------------|---------------|-------------------|------------|
| Solar PV | Modules, inverters, mounting structures, trackers | IEC 61215/61730, IP65/IP66, anti-PID | 8-20 weeks |
| Wind Turbines | Nacelles, blades, towers, control systems | IEC 61400 series, class-specific | 40-80 weeks |
| Transformers | Power transformers, distribution transformers | IEC 60076 SANS 780, oil/air insulated | 20-40 weeks |
| Switchgear | MV/HV switchgear, protection relays, SCADA | IEC 62271, SANS 10086 | 16-30 weeks |
| Cables | HV/MV/LV cables, fiber optic, control cables | SANS 1507/1429, armored, fire-rated | 8-16 weeks |
| Structural Steel | Towers, gantries, substation structures | Galvanized per SANS 121, structural steel specs | 12-20 weeks |
| Turbines (Thermal) | Steam/gas turbines, generators, HRSG | API 612/616, ASME, OEM specifications | 40-80 weeks |

## Common SO Tracking Requirements

| Category | SO Focus Areas | Critical SO Verification Points |
|----------|---------------|--------------------------------|
| Renewable Equipment | Manufacturing milestones | FAT, performance tests, grid compliance certificates |
| Transmission Equipment | Component scheduling | Type test reports, factory acceptance, warranty terms |
| Balance of Plant | Delivery coordination, compatibility | Interface specifications, control system integration |
| Grid Equipment | Grid code compliance testing | Utility acceptance, protection coordination studies |

## Common WO Types (Work Orders for Services)

| WO Category | Typical Services | Compliance Requirements |
|-------------|-----------------|------------------------|
| Civil Works | Foundations, access roads, control buildings | Structural inspection, concrete testing, fencing |
| Erection Services | Turbine erection, panel installation, stringing | OEM erection procedures, safety protocols |
| Grid Connection | Substation commissioning, protection testing | Grid code compliance, utility witness testing |
| Commissioning | System integration, performance testing | Performance guarantees, grid synchronization |
| O&M Services | Operations and maintenance, monitoring | OEM service agreements, SLAs, availability guarantees |
| Environmental | Environmental monitoring, rehabilitation | EIA conditions, water use licenses, rehabilitation bonds |

## Key Goods vs Services Distinctions for Energy & Power

### Goods (PO Items)
- Generation equipment with performance guarantees and grid compliance
- Balance of plant with interface specifications and control systems
- Transformers and switchgear with type testing and grid acceptance
- Solar modules with degradation warranties and IEC certification

### Services (WO/Service Order Items)
- Erection services with OEM supervision requirements
- Commissioning with performance testing against guarantees
- Grid connection with utility witness and compliance testing
- O&M with availability guarantees and response time SLAs

## Project-Specific Procurement Considerations
- **Solar PV:** Panel warranties (25+ year), inverter replacements, tracker maintenance contracts
- **Wind Turbines:** Long manufacturing lead times, transport logistics for blades/towers, crane requirements
- **Thermal Power:** Fuel supply agreements, emissions control equipment, cooling systems
- **Hydro:** Dam construction, penstock fabrication, turbine-generator assembly
- **Transmission:** Wayleaves/wayleave negotiations, crossing agreements, specialized conductor materials
- **Substations:** Protection coordination, SCADA integration, utility witness testing requirements
