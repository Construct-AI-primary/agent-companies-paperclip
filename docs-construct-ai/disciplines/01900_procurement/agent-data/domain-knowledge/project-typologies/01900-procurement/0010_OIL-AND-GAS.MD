# Oil & Gas Project Procurement Typology

## Sector Overview
Oil & gas projects encompass upstream (exploration, drilling), midstream (pipelines, terminals), and downstream (refining, petrochemicals). Procurement involves highly specialized equipment, stringent specification requirements, and complex service contracts.

## Common PO Types (Purchase Orders for Goods)

| PO Category | Typical Goods | Key Specifications | Lead Times |
|-------------|---------------|-------------------|------------|
| Drilling Equipment | Drill bits, casing, tubing, mud pumps | API 5CT, API 10A, NACE MR0175 | 8-20 weeks |
| Wellhead Equipment | Christmas trees, BOP systems, manifolds | API 6A, API 16C | 20-40 weeks |
| Pipelines and Fittings | Line pipe, flanges, valves, fittings | API 5L, ASME B31.3/B31.4/B31.8 | 12-26 weeks |
| Processing Equipment | Heat exchangers, separators, compressors | ASME VIII, API 12J, API 617 | 30-60 weeks |
| Instrumentation | Transmitters, analyzers, control systems | IEC 61508/61511, SIL-rated, ATEX/IECEx | 16-30 weeks |
| Materials | Carbon steel, stainless steel, special alloys | ASTM, ASME, API, NORSOK | 6-16 weeks |
| Electrical | Transformers, generators, switchgear | IEC 60076, API 541, hazardous area | 20-40 weeks |

## Common SO Tracking Requirements (Supplier Order Confirmations)

| Category | SO Focus Areas | Critical SO Verification Points |
|----------|---------------|--------------------------------|
| Critical Equipment | Vendor manufacturing schedule | FAT/IFAT schedules, inspection hold points, milestone dates |
| Standard Equipment | Supplier delivery confirmations | Delivery dates match PO, specifications confirmed |
| Materials | Mill delivery schedules | Heat numbers, material certificates, country of origin |
| Consumables | Ongoing supply agreements | Call-off schedules, delivery locations |

## Common WO Types (Work Orders for Services)

| WO Category | Typical Services | Compliance Requirements |
|-------------|-----------------|------------------------|
| Drilling Services | Directional drilling, mud logging, cementing | API standards, company drilling procedures |
| Intervention Services | Wireline, coiled tubing, snubbing operations | Well control certification, IWCF/WellCAP |
| Construction Services | Fabrication, installation, precommissioning | WPS/PQR qualifications, NDT certification |
| Commissioning Services | System integration, performance testing | OSHA/IOSH, company commissioning procedures |
| Maintenance Services | Equipment maintenance, calibration, inspection | OEM procedures, calibration certificates, competency |
| Camp/Life Support | Catering, accommodation, medical services | Company HSSE standards, hygiene certifications |

## Key Goods vs Services Distinctions for O&G

### Goods (PO Items)
- Long-lead equipment with FAT/IFAT requirements
- Materials with heat number and material certificate requirements
- Hazardous area equipment with ATEX/IECEx certification
- Items requiring third-party inspection (TPI) and release certificates

### Services (WO/Service Order Items)
- Well services requiring company representative certification
- Construction and fabrication with WPS/PQR requirements
- Commissioning and startup services requiring system handover documentation
- Logistics and transportation with dangerous goods certification

## Project-Specific Procurement Considerations
- **Brownfield Modifications:** Material matching, existing equipment compatibility, permits to work
- **Greenfield Projects:** Long-lead equipment scheduling, construction workforce logistics
- **Offshore Projects:** Maritime transport, offshore supply vessels, marine certifications
- **Remote Locations:** Logistics chains, local content requirements, customs clearance
