# 00877 Sundry AI-Native Operations Prompt Template

## Overview

This AI-Native development prompt guides coding agents to implement miscellaneous project support features, administrative functions, office management, and temporary task coordination.

**This prompt is for OpenClaw coding agents operating in dev mode.**

## Implementation Phases

### Phase 1: Foundation — Sundry Data Architecture
- [ ] Classify data: text-native (office records, admin requests, temporary task logs)
- [ ] Implement task tracking pipeline

### Phase 2: Document Generation Pipeline
- [ ] Create template library — Administrative forms, meeting minutes, task assignment records

### Phase 3: Multi-Agent Sundry Orchestration
- [ ] Agents: Sundry Coordinator, Admin Support Agent, Task Assignment Agent

### Phase 4: Intelligence
- [ ] Implement resource allocation optimization for ad-hoc tasks

### Phase 5: Natural Language Interface
- [ ] "Show me all open administrative tasks for Project X"

### Phase 6: Compliance & AI Safety Boundaries
- [ ] Non-delegable: Financial approvals require management authorization
- [ ] Never make decisions requiring specialist knowledge
- [ ] Always document task outcomes

## Behavioral Rules
1. **Delegation:** Never make decisions requiring specialist knowledge; escalate unclear requirements
2. **Accuracy:** Do not invent administrative records or meeting outcomes
3. **Provenance:** Always track the source and assignee of tasks
4. **Authority Rule:** Never override management decisions
