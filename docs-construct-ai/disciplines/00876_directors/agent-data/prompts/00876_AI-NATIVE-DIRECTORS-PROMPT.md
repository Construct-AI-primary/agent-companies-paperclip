# 00876 Directors AI-Native Operations Prompt Template

## Overview
This AI-Native development prompt guides OpenClaw coding agents to implement executive governance, strategic oversight, and board support features. **This prompt is for OpenClaw coding agents operating in dev mode.**

## Implementation Phases

### Phase 1: Foundation — Executive Data Architecture
- [ ] Classify data: text-native (board packs, strategy documents) or structured (financial dashboards, KPI reports)
- [ ] Implement board pack generation pipeline

### Phase 2: Document Generation
- [ ] Template library — Board presentations, strategic reports, annual reports, executive summaries

### Phase 3: Multi-Agent Executive Support
- [ ] Agents: Executive Assistant, Board Analytics Agent, Risk Dashboard Agent

### Phase 4: Executive Intelligence
- [ ] Board pack automation, KPI dashboard generation, strategic trend analysis

### Phase 5: AI Safety Boundaries
- [ ] Non-delegable: Board decisions require human authorization
- [ ] AI must disclose when strategic data assumptions unverified

## Automation Spectrum
| Level | Examples |
|-------|----------|
| Full Automation | KPI report formatting, board pack assembly, meeting minutes drafting |
| Augment | Strategic analysis support, risk assessment compilation |
| Human-Led | Board decisions, strategic approvals, executive appointments |
