## Platforms

This project supports **desktop**, **mobile**, and **web** platforms to ensure comprehensive perimeter security coverage across all operational contexts.

### Desktop Platform
- **Target Users:** Security operations center personnel, site security managers, system administrators
- **Key Features:** Zone mapping, sensor configuration, alert dispatch, incident correlation, compliance reporting
- **UI/UX Focus:** DomainForge AI / DevForge AI

### Mobile Platform
- **Target Users:** Field security guards, site supervisors, mobile patrol officers
- **Key Features:** Remote sensor monitoring, intrusion alert acknowledgment, video verification, offline mode
- **UI/UX Focus:** DevForge AI

### Web Platform
- **Target Users:** Security directors, compliance officers, facility managers
- **Key Features:** Zone status dashboard, sensor health board, audit trail, incident reports
- **UI/UX Focus:** DevForge AI / DomainForge AI
