# HLTH-FIT-001: Fitness for Work Foundation — Assessment Framework, Medical Standards & Data Model

## Description

Establish the foundational architecture for the fitness for work and health assessment system. This issue covers the design of the assessment framework, medical fitness standards, worker health data model, and assessment workflow schema.

**Phase:** 1 — Foundation
**Priority:** Critical
**Story Points:** 8
**Estimated Duration:** 4 weeks
**Assignee:** domainforge-ai
**Company:** domainforge-ai
**Due Date:** 2026-06-30

**Blocks:** HLTH-FIT-002, HLTH-FIT-003, HLTH-FIT-004, HLTH-FIT-005

## Goals

1. Design comprehensive fitness for work system architecture
2. Define assessment framework (pre-employment, periodic, return-to-work, functional)
3. Establish medical fitness standards and categorization
4. Design worker fitness data model with restriction tracking
5. Define data models and database schema

## Acceptance Criteria

- [ ] Fitness for work system architecture documented
- [ ] Assessment types defined: pre-employment, periodic, return-to-work, functional, drug & alcohol
- [ ] Fitness categories defined: fully fit, fit with adjustments, temporarily unfit, permanently unfit
- [ ] Medical restriction framework with expiry tracking
- [ ] Worker fitness data model with longitudinal tracking
- [ ] Database schema with RLS policies
