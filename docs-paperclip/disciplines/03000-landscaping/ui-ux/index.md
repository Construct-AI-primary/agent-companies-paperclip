**Version:** 1.0  
**Last Updated:** 2026-05-04

## Overview

This UI/UX specification defines the user interface and user experience for the Landscaping discipline (03000) within the Paperclip ecosystem. The landscaping system covers landscape design, planting plans, irrigation management, hardscaping, site preparation, maintenance scheduling, and environmental enhancement across desktop, mobile, and web platforms.

## Color Scheme

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Primary | Landscape Green | `#2e7d32` | Headers, primary buttons, navigation |
| Secondary | Earth Brown | `#5d4037` | Hardscape elements, terrain features |
| Success | Growth Green | `#4caf50` | Healthy plants, completed tasks |
| Warning | Drought Amber | `#ff8f00` | Irrigation alerts, pending actions |
| Info | Sky Blue | `#0288d1` | Water systems, weather data |
| Accent | Bloom Pink | `#c2185b` | Flowering plants, seasonal highlights |

## Screen States

Every landscaping screen supports four states:
1. **Loading** — Skeleton screens with nature-themed placeholders
2. **Empty** — "No plants" / "No designs" illustrations with guidance
3. **Error** — Error state with retry action and fallback procedures
4. **Data** — Full data presentation with visual plant/design elements

## Design Principles

1. **Visual Planning** — CAD-like design tools for planting plans and hardscape layouts
2. **Seasonal Awareness** — Calendar-based views for planting seasons, maintenance cycles
3. **Environmental Context** — Integration with soil, climate, and site condition data
4. **Field-First** — Mobile tools optimized for on-site surveys and inspections
5. **Sustainability** — Water conservation, native planting, and biodiversity metrics
